# The Rock Paper Scissors Game!

This is the repo for the Rock, Paper, Scissors game!

## Learning Outcomes

Through this project, I have learned:
* Using Javascript in the console to play a game of rock paper scissors
* Used conditionals, creating functions and string manipulation in Javascript

## Link to Live Project

https://bagasprimanto.github.io/rock-paper-scissors/


## How to use:
Currently you can only play the rock paper scissors game for 5 rounds on the console log.

Enter your choice (Rock, Paper or Scissors) into the prompt box. The site will prompt 5 times.

To see the results of the game, open your web browser's console.

Enjoy!
